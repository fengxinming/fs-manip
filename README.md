# fs-manip

> adds promise support to the fs methods.

## Installation

```bash
$ npm install fs-manip --save
```

## API

### Async

* copy
* emptyDir
* exists
* makeDir
* move
* outputFile
* remove

### Sync
  
* copySync
* emptyDirSync
* existsSync
* makeDirSync
* moveSync
* outputFileSync
* removeSync
